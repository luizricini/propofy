# propofy

Micro-SaaS de geração de propostas comerciais e orçamentos com IA para prestadores de serviço brasileiros.

- **Docs:** decisões de arquitetura em [`docs/adr/`](docs/adr/) e escopos com critérios de aceite em [`docs/scopes/`](docs/scopes/). Leia [`CLAUDE.md`](CLAUDE.md) antes de contribuir.
- **Stack:** Next.js (App Router) + Supabase + Vercel. Detalhes na [ADR-001](docs/adr/ADR-2026-15-07-stack-e-infraestrutura.md).

## Desenvolvimento

```bash
pnpm install
pnpm dev
```

Requer Node 22+ e pnpm.
